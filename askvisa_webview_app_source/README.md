# ASKVISA Flutter WebView App Setup Guide
Please refer to the flutter_setup_guide.md provided in the chat artifacts for full instructions on setting up this app!

## Quick Start
1. Run `flutter create askvisa_app`
2. Replace `lib` and `pubspec.yaml` with the contents of this folder.
3. Configure `AndroidManifest.xml` (See Setup Guide).
4. Run `flutter build apk`.
